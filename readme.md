# logo-if

Desenha a logo do Instituto Federal(IF) através do módulo *turtle* da linguagem *Python*.

## Como usar

1. Clone o repositório : `git clone https://github.com/gabr-01/logo-if.git`
2. Execute o arquivo *logo_if.py*
